# 5. gyakorlat
## Feladat: Autók nyilvántartása
Készítsünk menüvezérelt programot autók nyilvántartására!
Lehessen az autókat 
-	felvinni, 
-	listázni,
-	keresni,
-	módosítani,
-	törölni.

Egy autónak van 
-	rendszáma, 
-	típusa,
-	színe,
-	ára.

Az autókat egy rendszám szerint kereshető konténerbe tegyük (HashMap, TreeMap).
